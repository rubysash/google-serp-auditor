How It Works (Step by Step)

1
Find a Business on Google Maps

Search for any business on Google Maps (like "Joe's Pizza Dallas"). Click on the business listing to open its details page.

2
Copy the URL

Copy the web address from your browser's address bar. It should look like: https://www.google.com/maps/place/...

3
Paste and Extract

Paste the URL into our tool and click "Extract Data". We'll automatically pull out the business name, location coordinates, and unique identifiers.

4
Choose Your Intelligence Level

Select which APIs to use: Place Details (address, phone, website) and/or Knowledge Graph (entity recognition, authority data).

5
Get Instant Intelligence

Click "Enrich Data" and watch as we gather comprehensive business information and generate 100+ verification and audit links in seconds.

6
Use the Results

Review the JSON data and click through the Knowledge Links tab to audit the business's online presence, SEO, and competitive position.

Quick Setup Guide

1 Create Google Cloud Account

Go to console.cloud.google.com and create a new project called "maps-intelligence"

2 Enable Required APIs

In the API Library, search for and enable:

  • Places API - For business details (address, phone, website)
  • Knowledge Graph Search API - For entity recognition and authority data
3 Create API Key

Go to Credentials → Create Credentials → API Key. Copy the key and add it to your .env file as:

GOOGLE_MAPS_API_KEY=your_api_key_here
4 Set Up OAuth (Optional)

For additional security, you can restrict API access to specific Google accounts. See the OAuth setup documentation in the project files for details.

Pro Tips for Maximum Efficiency

Workflow Optimization

  • Batch Processing: Open multiple browser tabs with different business listings, then process them one by one for maximum speed.
  • Use Search Terms: Start with the search feature to find businesses systematically (e.g., "plumbers in dallas", "restaurants in austin").
  • Save the JSON: Copy the JSON data from each lookup to build comprehensive client reports and databases.

What to Look For

  • Missing Data: If phone, website, or address show "Not available", that's an immediate optimization opportunity for your client.
  • Review Count: Low review counts or ratings indicate reputation management opportunities.
  • Knowledge Graph Status: If no KG entity is found, the business has potential for authority building and entity optimization.

Common Issues & Solutions

Cause: API key not configured or lacks permissions

Solution:

  • Check that your GOOGLE_MAPS_API_KEY is set in the .env file
  • Verify Places API and Knowledge Graph API are enabled in Google Cloud Console
  • Ensure billing is enabled on your Google Cloud project

Cause: Business location is very new or not well-established in Google Maps

Solution:

  • Try searching for the business by name instead of using coordinates
  • Check if the coordinates are accurate by pasting them into Google Maps
  • Some very new businesses may not have Place IDs yet - this is normal
